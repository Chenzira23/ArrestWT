# Changelog

All notable changes to this project will be documented in this file.

## [0.0.2] - 2025-08-24

### Added

- App icons in the following formats: LDPI, MDPI, TVDPI, HDPI, XHDPI, XXHDPI, XXXHDPI
- Add card page
- Create file function in DataUtils for creating JSON files
- Edit data function in DataUtils for editing JSON files
- Account adding, editing, and deleting
- Card adding, editing, and deleting
- Account editing popup
- Card editing popup
- Checkbox setting UI component
- Changed popup container color to the same as background color for readability with themes

### Changed

- Moved tags page button to performance page
- Changed some icons for pages
- All card related ui components are now in component/cards
- All setting related ui components are now in component/settings
- All user related data is now managed in the files directory
- Hidden text setting popups now have a rounded text field outline
- Account, Card, Org, and Person data types can now have tags
- Added to Roadmap

### Changed
